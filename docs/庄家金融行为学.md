# 庄家金融行为学 — 拉升前行为模式与信号设计

> 本文档基于对 ERAUSDT、ESPORTSUSDT、REUSDT、BANKUSDT 四个近期(2026-07)大涨币的实证分析，归纳庄家拉升前的可观测行为模式，并据此设计领先信号算法。
>
> 核心立场：**信号必须捕捉"拉升前"而非"拉升中"**。旧合约策略(OI/热度/费率)是"已涨完"的结果指标，表现为追高跑输基准；本设计转向捕捉庄家吸筹/洗盘/收敛的领先行为序列。

---

## 第一部分：庄家金融行为学理论

### 1.1 庄家操盘的完整生命周期

大资金（庄家）拉升一个币通常经历四个阶段，顺序固定：

```
① 建仓(吸筹)  →  ② 洗盘(清洗浮筹)  →  ③ 末筹收敛  →  ④ 拉升起飞  →  ⑤ 出货
   分批吃货       制造波动震出局       低调收尾       量价齐升       派发
   (领先)         (领先)              (领先/临界)     (结果)         (结果)
```

我们想要的入场点在 **③末筹收敛、即将进入④的临界点**——这是左侧入场。④之后是追高，⑤是接盘，都要避免。

### 1.2 四个行为特征的动机与可观测性

#### ① 多脉冲买入（建仓期）——"放量但不涨"是核心

**动机**：庄家不会一次买完（暴露意图、推高成本），而是分批吃货。吃货时主动压价：吃掉上方卖单但不主动上推，避免引起注意。

**可观测背离**：成交量脉冲放大，但价格没创新高、涨幅远小于量增幅。
- **量价背离度** = 量增幅 / 价格涨幅。背离越大（量大涨小），吸筹嫌疑越强。
- **关键判据**：放量本该推高价格；若放量却不涨甚至小跌，说明有卖盘在压制——要么庄家压价吸筹，要么上面有抛压被悄悄吃掉。

#### ② 高波动洗盘——"跌了但收回来"是核心

**动机**：建仓末期制造剧烈波动，把散户震出局捡恐慌盘，降低自身成本，测试抛压。

**可观测**：振幅大、长下影线、A字型（急跌急拉）、单日大跌后快速收回。
- **洗盘 vs 趋势跌的区别**：洗盘会收回来（V型），趋势下跌不收回。
- **回收率** = min(close) / min(low)，或下影线长度占比。洗盘特征是"探底收回"。
- **关键判据**：急跌(24h<-8%)后3日内收盘价回到跌前高位的95%以上 = 洗盘；否则是趋势下跌（要避开）。

#### ③ 隐蔽囤筹（末筹收敛）——"波动率收敛 + 地量"是核心

**动机**：洗盘后期浮筹出清，庄家低调收尾。暴风雨前的宁静，最佳左侧入场点。

**可观测**：波动率从高**收窄**（状态转换，非持续低）、成交量萎缩到极致（地量）。
- **波动率收敛** = 近3日amp / 前期amp < 0.6。这是"高波动→低波动"的**状态转换**。
- **关键**：是状态转换，不是"一直低波动"。当前 amp<8 算法要求"全程低波动"，恰恰过滤掉了最有爆发力的"洗盘后收敛"型强庄币。

#### ④ 快速拉升起飞——结果指标，不追

**表现**：量价齐升、突破前高。此时浮筹已集中，少量资金就能拉动。
- 这是确认/出场信号，不是入场信号。旧策略错就错在把④当入场点。

### 1.3 对当前"横盘暗筹"算法的根本反思

| 维度 | 当前算法 amp<8 | 庄家真实行为 |
|------|---------------|-------------|
| 逻辑 | 要求**全程**7日低波动 | 拉升前是**高波动洗盘→收敛** |
| 匹配的庄 | 慢庄、老币慢吸 | 强庄、脉冲式洗盘后拉升 |
| 结果 | 只抓低波动币，过滤掉强庄币 | 强庄币（20倍行情）全漏 |

**BANKUSDT 6-7月20倍行情完全漏掉**的根因：7日窗口往回看还含洗盘波动，amp 长期 >8%（实测38%），被门槛过滤。这证明"全程低波动"假设是错的——强庄币拉升前恰恰是高波动洗盘。

## 第二部分：四币实证模式分析

> 数据：2026-04-30 ~ 2026-07-22，1h K线。对每个币找最近一次大涨（24h涨幅最大），分析其前14天的庄家行为。

### 2.1 ERAUSDT — 模式A：低点吸筹脉冲式

- **涨幅**：7天 +70.1%，起飞点 2026-07-21 20:00（24h +71.6%）
- **量价背离脉冲**：起飞前2天连续2个"放量不涨"脉冲
  - 07-19：量2.5x中位，跌-5.4%
  - 07-20：量2.2x中位，跌-2.8%
  - → 典型低点吸筹（放量但压价）
- **洗盘**：07-17 多次急跌(-8%)，部分未收回
- **波动率收敛**：amp序列[3.8, 6.1, 8.3, 11.3] → **扩张**（未收敛），起飞前波动增大
- **地量**：起飞前量比43x → **非地量，已放量启动**
- **走势**：起飞前5日 0.077→0.0612(先跌到低点)→0.105(起飞)
- **模式归纳**：**缓跌洗盘 → 低点放量不涨吸筹(2脉冲) → 起飞**。波动扩张型，无收敛。

### 2.2 ESPORTSUSDT — 模式B：洗盘收敛式（最经典）

- **涨幅**：起飞 2026-07-17（24h +97.2%），前7天 +44.7%
- **量价背离脉冲**：2个放量不涨脉冲
  - 07-08：量2.4x，跌-15.3%（巨量杀跌式吸筹）
  - 07-10：量3.1x，跌-5.9%
- **洗盘**：窗口内 **80次**急跌(24h<-8%)，其中46次收回 → **剧烈高频洗盘**，V型特征明显
- **波动率收敛**：amp序列[22.5, 15.9, 12.3, 12.3] → 末段/首段=**0.55 ✅收敛**（最典型）
- **地量**：量比12.7x → 非地量，已放量
- **走势**：起飞前5日 0.016→0.01333→0.01443→0.02846（低位横盘后爆发）
- **模式归纳**：**剧烈洗盘(80次急跌) → 波动率收敛(0.55) → 起飞**。教科书式洗盘后收敛。

### 2.3 REUSDT — 模式C：超跌反弹式（弱信号）

- **涨幅**：起飞 2026-07-22（24h +24.4%），但7天 -7.7% → 刚反弹
- **量价背离脉冲**：4个，其中2个是巨量杀跌
  - 07-17：量**11.1x**，跌-12.5%（巨量杀跌，非吸筹）
  - 07-18：量3.0x，跌-13.1%
  - 07-21：量2.6x，涨+1.6%（这个是吸筹）
- **洗盘**：59次急跌，仅25次收回（收回率低 → 偏趋势跌而非洗盘）
- **波动率收敛**：amp序列[5.2, 15.1, 14.8, 15.8] → 未收敛
- **走势**：起飞前5日 0.458→0.3813→0.3691→0.4593（跌深后反弹）
- **模式归纳**：**巨量杀跌 → 企稳 → 超跌反弹**。信号弱，可能是反弹而非主升。07-21 的"放量小涨"才是可疑吸筹点。

### 2.4 BANKUSDT — 模式D：收敛爆发式（最强）

- **涨幅**：7天 +166.5%，起飞 2026-07-19（24h +166.6%），振幅702%
- **量价背离脉冲**：07-12 量2.6x 跌-2.6%（1个吸筹脉冲）
- **洗盘**：4次急跌，**4次全部收回**（收回率100%）
  - 07-09 -8.5% ✅、07-17 -18.6% ✅、07-18 -13.1% ✅、07-18 -10.8% ✅
  - → **完美V型洗盘**（每次急跌都收回 = 强庄控盘）
- **波动率收敛**：amp序列[12.4, 7.1, 3.9, 34.7] → 前3段收敛到**3.9%极低值**，第4段爆发
  - 收敛到3.9%是爆发前最明确的信号
- **地量**：量比13.57x → 非地量
- **走势**：起飞前5日 0.0429→0.0463→0.0608→0.0495→0.0713→0.1901（盘整后爆发）
- **模式归纳**：**V型洗盘(全收回) → 波动率收敛到极致(3.9%) → 爆发**。

### 2.5 四币模式横向对比

| 维度 | ERA(模式A) | ESPORTS(模式B) | RE(模式C) | BANK(模式D) |
|------|-----------|--------------|-----------|------------|
| 量价背离脉冲 | 2个(低点吸筹) | 2个(杀跌吸筹) | 4个(混合) | 1个 |
| 洗盘收回率 | 部分收回 | 46/80收回 | 25/59收回 | **4/4全收回** |
| 波动率收敛 | ❌扩张 | ✅0.55 | ❌未收敛 | ✅收敛到3.9% |
| 起飞强度 | +70% | +97% | +24%(弱) | +166%(最强) |
| 启动前地量 | ❌放量 | ❌放量 | ❌放量 | ❌放量 |

### 2.6 实证得出的关键结论

1. **量价背离脉冲是四币共同特征**（ERA 2、ESPORTS 2、RE 4、BANK 1）→ 这是吸筹最可靠的领先信号，应作为算法**核心门槛**。

2. **洗盘V型是强庄币的标志**：收回率越高，庄家越强。BANK(100%收回)和ESPORTS(57%收回)涨幅最大；RE(42%收回)涨幅最小。**洗盘收回率可作为强度评分**。

3. **波动率收敛只对部分模式有效**：ESPORTS(洗盘后收敛)和BANK(收敛到极致)有明显收敛；ERA(扩张)和RE(未收敛)没有。**收敛应作为加分而非门槛**，否则漏掉ERA这类模式。

4. **地量不是起飞前特征**：四币起飞前量比都 >6x（已放量启动）。"地量"是起飞前2-3周的特征，起飞前1-2天已是放量。**不应用地量作为起飞前入场门槛**——应捕捉更早的"吸筹脉冲"。

5. **"全程低波动"假设错误**：四币起飞前2周都经历了高波动洗盘。amp<8会过滤掉全部4个币 → 旧算法根本抓不到强庄币。

6. **不同模式需要差异化识别**，但共性核心是：**量价背离脉冲(吸筹) + 洗盘V型(强庄) + 处于低位(未大涨)**。

## 第三部分：综合算法设计

### 3.1 设计原则

基于实证，信号应检测**行为序列**而非单一持续状态：

```
[洗盘发生过且收回]  AND  [近期有量价背离脉冲(吸筹)]  AND  [处于近期相对低位(未大涨)]
        ↑                      ↑                              ↑
   急跌后V型收回          放量但涨幅小              价格在近2周中低位
```

放弃"全程低波动"和"地量"假设，改为检测**洗盘+吸筹+低位**三要素。

### 3.2 信号特征定义（v2 算法）

对每个币计算（基于1h K线，窗口近14天336根）：

```
1. 量价背离脉冲 vol_div_pulse:
   按日聚合窗口。若存在某日: 日量 > 窗口日量中位 × 2  AND  日涨幅 ∈ [-15%, +3%]
   → 命中吸筹脉冲。计数 pulse_n。

2. 洗盘V型回收 washout_recover:
   扫描窗口内所有 24h跌幅 < -8% 的事件。
   若3日内收盘价回到跌前高的96%以上 → 判为洗盘收回。
   wash_recov_rate = 收回次数 / 急跌总次数。
   且要求急跌事件数 wash_n >= 1 (洗盘发生过)。

3. 低位位置 position_low:
   当前价在近14天[最低,最高]区间的位置 < 0.4 (处于相对低位，未大涨)。
   pos = (close - low14) / (high14 - low14)。

4. 波动率收敛 vol_converge (加分，非门槛):
   近3日amp / 前7日amp < 0.7 → 收敛加分。

5. 资金费 frate (过滤):
   frate <= 0.02% (不拥挤做多，空头燃料已备)。

6. 未在大涨中 not_pumping (过滤):
   近24h涨幅 < 8% (避免追在起飞中)。
```

### 3.3 入选门槛

全部满足才入选：
```
pulse_n >= 1              至少1个量价背离吸筹脉冲
wash_n >= 1 AND wash_recov_rate >= 0.4   洗盘发生过且收回率≥40%(强庄特征)
pos < 0.4                 处于近14天相对低位
frate <= 0.02             资金费不拥挤
chg24 < 8                 未在追高大涨中
chg5d < 10                5日未大涨(避免追高)
```

### 3.4 强度评分（满分约100）

| 维度 | 满分 | 分档 |
|------|------|------|
| 洗盘收回率 wash_recov_rate（越强庄越好）| 30 | ≥0.8→30, ≥0.6→20, ≥0.4→10 |
| 吸筹脉冲数 pulse_n（越多越好）| 25 | ≥3→25, ≥2→18, ≥1→10 |
| 低位程度 pos（越低越好）| 20 | <0.2→20, <0.3→13, <0.4→7 |
| 波动率收敛 converge（收敛加分）| 15 | amp3/amp7<0.5→15, <0.7→8, 否则0 |
| 资金费 frate（越负越好）| 10 | <-0.03→10, <-0.01→6, ≤0.02→3 |

`score = wash_recov + pulse + pos + converge + frate`，降序取 Top 20。

### 3.5 与旧算法(横盘暗筹v1)的关键差异

| 维度 | v1 横盘暗筹 | v2 庄家行为序列 |
|------|-----------|--------------|
| 核心假设 | 全程7日低波动 | 洗盘+吸筹行为序列 |
| 门槛 | amp<8 | 量价背离脉冲+洗盘收回+低位 |
| 能否抓强庄币 | ❌(amp过滤掉) | ✅(检测洗盘而非要求低波动) |
| BANK 20倍 | 漏掉 | 应能抓(100%洗盘收回+吸筹脉冲+低位) |
| 波动率收敛 | 门槛(必须) | 加分(可选，兼容ERA扩张型) |
| 地量 | 隐含 | 放弃(实证表明起飞前已放量) |

### 3.6 验证计划

1. **单币验证**：先在 BANKUSDT/ERAUSDT 上验证 v2 能否抓到各自起飞前的入场点。
2. **全市场回测**：359币×3个月，无look-ahead，退市币纳入，对比基准。要求：超额收益为正、胜率>45%、风险收益比>1。
3. **与v1对比**：v2 应在强庄币(BANK/ESPORTS)上显著优于 v1，且整体不退化。

## 第四部分：验证结果与最终算法决策

### 4.1 单币验证结果（4币）

| 币 | 模式 | v2命中 | 表现 | 结论 |
|----|------|--------|------|------|
| BANKUSDT | D收敛爆发 | 15个全盈利 | 5天+13~23%，最大跌-0.3~-5.5% | ✅完美抓到起飞前12天 |
| ESPORTSUSDT | B洗盘收敛 | 后期命中抓起飞 | +30~40%(后期)，早期假信号 | ✅后期能抓，早期偏早 |
| ERAUSDT | A阴跌吸筹 | 漏掉 | wash_rate 0.04 过低被过滤 | ❌模式不匹配 |
| REUSDT | C超跌反弹 | 误判全亏 | -10~15%(持续下跌) | ❌下跌被误当洗盘 |

**关键发现**：v2 能稳定抓住 BANK/ESPORTS(洗盘型强庄)，但 ERA(阴跌吸筹)和 RE(超跌反弹)事前难区分——都表现为下跌+低位+脉冲，ERA后来起飞、RE继续跌。这是根本难点：事前无法预知哪个下跌会反转。

### 4.2 全市场回测结果（359币×95天）

| 算法 | 信号数 | 5天PnL均值 | 胜率 | 超额(vs基准) |
|------|--------|-----------|------|------------|
| 基准(随机) | 10740 | -0.13% | 46.4% | — |
| **v1 横盘暗筹** | 416 | +0.66% | 49.0% | **+0.79%** ✅ |
| v2 庄家行为序列(松) | 1644 | -1.18% | 40.3% | -1.05% ❌ |
| v2 收紧(wash≥0.5/pulse≥2) | 211 | +0.03% | 44.5% | +0.13% |
| v2 高分子(≥50分) | 189 | +0.64% | 46.6% | ~+0.74% |

### 4.3 诚实结论

1. **v2(庄家行为序列)全市场回测不优于 v1**：松门槛-1.05%(失败)，收紧后+0.13%，高分+0.74%——均不超过 v1 的 +0.79%。
2. **这是过拟合陷阱的实证**：4个币都是“大涨幸存者”，从它们归纳的模式存在幸存者偏差。大量下跌币也有“脉冲+低位+洗盘”特征但继续跌，事前无法区分。
3. **v2 的价值是互补而非替代**：v2 高分子能抓 v1 漏掉的强庄币(BANK/ESPORTS)，覆盖不同模式。但单独 alpha 不够。

### 4.4 最终算法决策

**生产采用 v1(横盘暗筹)为唯一上线信号**，理由：
- 全市场回测超额 +0.79%，胜率49%，风险收益均衡，是唯一通过验证的信号。
- 简单且稳健(门槛 amp<8 对全市场普适，不易过拟合)。

**v2 作为研究与互补层保留**，不上线，理由：
- 全市场 alpha 不足，且参数对4币敏感(过拟合风险)。
- 4币行为分析的价值在于**模式认知**(识别强庄 vs 弱庄 vs 趋势跌)，用于人工复核信号，而非自动化入场。

**改进方向(未来)**：
- “事前区分洗盘与趋势跌”是未解难题，需引入链上数据(筹码集中度/交易所流入)等异源信号，而非单靠K线。
- v2 的“洗盘收回率”可作为 v1 信号的**复核指标**：v1 命中后，若该币洗盘收回率高(强庄)，加仓权重。
